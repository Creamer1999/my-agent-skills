# My Agent Skills Library

A curated collection of **75 AI agent skills** for [Manus](https://manus.im) — covering marketing, content creation, SEO, CRO, strategy, engineering, design, and client services.

All skills were security-scanned with ClamAV (3.6M+ virus signatures) and checked for prompt injection, data exfiltration, and hidden executables before installation.

---

## How to Import a Skill into Manus

1. Go to **Skills → + Add → Import from GitHub**
2. Paste the URL of any skill folder below, e.g.:
   `https://github.com/elastaboy99/my-agent-skills/tree/main/mkt-copywriting`
3. Click **Import** — done!
